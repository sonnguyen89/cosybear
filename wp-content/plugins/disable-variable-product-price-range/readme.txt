== Description ==
disable the variable product price range. This usually looks like $100-$999. With this snippet you will be able to hide the highest price, plus add a “From: ” in front of the minimum price.
 
it may be best to not show your customers the highest possible price. In fact, if you go over to Apple’s website, you will see them using the “From” structure in their prices. Like the old way better?
